Hardware Abstraction Model

The following figure shows the CTR hardware abstraction model.

Hardware Abstraction Model
 

 

 

The CTR Peripheral Library provides interface routines to interact with the blocks shown in the previous diagram. 

 

Block 
Description 
Counter mode configuration 
APIs to set the counter block modes. 
Counter parameter configurations 
APIs to set the counter block parameters specific to the set mode. 
Trigger Set 
APIs to set the trigger sources to latch counter values. 
Latch settings 
APIs to set the latch modes to hold counter values upon trigger.
