EBI_MEMORY_SIZE Enumeration
C
typedef enum {
  CSNOTUSED,
  MEMORY_SIZE_64KB,
  MEMORY_SIZE_128KB,
  MEMORY_SIZE_256KB,
  MEMORY_SIZE_512KB,
  MEMORY_SIZE_1MB,
  MEMORY_SIZE_2MB,
  MEMORY_SIZE_4MB,
  MEMORY_SIZE_8MB,
  MEMORY_SIZE_16MB
} EBI_MEMORY_SIZE;
Description

Memory Size for Chip Select 

This enumeration selects memory size for Chip Select.

Members
Members 
Description 
CSNOTUSED 
Chip Select Not Used 
MEMORY_SIZE_64KB 
64 KB (smaller memories alias within this range) 
MEMORY_SIZE_128KB 
128 KB 
MEMORY_SIZE_256KB 
256 KB 
MEMORY_SIZE_512KB 
512 KB 
MEMORY_SIZE_1MB 
1 MB

4 MB 
MEMORY_SIZE_8MB 
8 MB 
MEMORY_SIZE_16MB 
16 MB 
Remarks

The caller should not rely on the specific numbers assigned to any of these values as they may change from one processor to the next.
