Running the Demonstration

Do the following to run the demonstration:

  1. If you have not already done so, use the BM64_bootloader application to give the BM64 module a unique device name.

 

 

  1. Compile the application selecting the appropriate MPLAB X IDE project configuration based on whether or not you are using the PIM (refer to Building the Application for details).
  2. Program the target device, and run the application. After the program starts, the name of the application should appear on the display and the connection icon in the top-right corner will be white (advertising).

 

 

  1. The mobile application, BLE Scanner, by Bluepixel Technologies LLP, is used in this demonstration. This application is available as a free download from iTunes. Locate and download this application to your smartphone.

Note: 
If you are unable find this application for your smartphone, there are several other BLE scanner applications available for the iPhone, which can be used with this MPLAB Harmony application. The functionality will be the same, but the screens may appear somewhat different from those shown here. 
  1. Start the BLE Scanner app. Once opened, it will begin scanning for BLE devices.

 

 

  1. Select the device name that you set up for your BM64 in Step 1 (running the BM64_bootloader program) and tap Connect. The connection icon in the top right corner of the LCD should change from gray to bright blue.

 

 

  1. A list of services is displayed.

 

 

  1. Select the service named CUSTOM SERVICE, with the UUID starting with 4953 and ending in E455. This service matches the Transparent Service UUID set up in the BLE section of the UI tool.

 

 

  1. A list of characteristics are displayed.

 

 

  1. If necessary, scroll down and tap the characteristic labeled Notify. Tap the Updating ? icon to the right, which will display the following screen (ensure that the slider switch to the right of Notify is enabled (set to the right position). Tap the white less than (<) symbol in the upper left corner of the screen to return to the previous screen).

 

 

  1. On the PIC32 Bluetooth Audio Development Kit, press one of the push buttons, SW1 through SW5.

 

 

  1. The text string 'Button SWx pressed' (where 'x' is the number of the button you pressed) should appear on your smartphone (in this example, SW4 was pressed):

 

 

  1. To send a text string from your smartphone back to the BM64, tap Write in the BLE Scanner screen:

 

 

  1. On the next screen, tap Write Value:

 

 

  1. When the popup appears, tap Text.

 

 

  1. Enter a message in the text box up to 20 characters in length, such as 'Sending text to BM64, and then tap Write.

 

 

  1. The message you entered should appear in the LCD screen of the PIC32 Bluetooth Audio Development Kit.

 

 

  1. If you send additional messages to the BM64 (always less than or equal to 20 characters in length), the previous messages will scroll up, and the newest message will appear at the bottom of the screen.
