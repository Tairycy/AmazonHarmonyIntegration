SAMPLE_MODULE_TIMEOUT Macro
C
#define SAMPLE_MODULE_TIMEOUT 100
Description

Sample Module, Timeout 

This macro defines how many milliseconds the sample module will block, waiting to obtain a mutex, when accessing its internal shared resources. This is only utilized in interrupt-driven modes, but needs to be defined to build correclty.

Remarks

Used by OSAL_MUTEX_Lock.
