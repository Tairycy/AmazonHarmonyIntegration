PLIB_MCPWM_SecondarySpecialEventTriggerSetup Function
C
void PLIB_MCPWM_SecondarySpecialEventTriggerSetup(
    MCPWM_MODULE_ID index, 
    uint16_t compare_value, 
    MCPWM_TRIGGER_DIVIDER postscaler_value
);
Description

This function performs following setup functions for Secondary Special Event Trigger

  1. Sets the compare match value
  2. Sets the Secondary Special Event Trigger Output Postscaler
 

Preconditions

The MCPWM Module must be disabled before calling this function.

Parameters
Parameters 
Description 
index 
Identifier for the device instance to be configured
 
compare_value 
16 bit Compare value (in Secondary MCPWM timer counts) which is compared against Secondary Master Time Base to generate a match event. 0<=compare_value<=65535
 
postscaler_value 
Trigger Post scaler value
 
Returns

None.

Remarks

This feature may not be available on all devices. Please refer to the specific device data sheet to determine availability or use PLIB_MCPWM_ExistsSecondarySpecialEventTrigger in your application to determine whether this feature is available.

Example
Copy Code
// Where MY_MCPWM_INSTANCE, is the MCPWM instance selected for use by the
// application developer.
// Compare match event occurs at Secondary MCPWM Timer (SMTMR) = 10000
// Secondary Special Event Trigger is generated at every 4th compare match event.

PLIB_MCPWM_SecondarySpecialEventTriggerSetup(MY_MCPWM_INSTANCE , 10000 , 4 );
