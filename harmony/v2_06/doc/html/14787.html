L2CAP_HCI_PAGE_SCAN_REPETITION_MODE 
brief L2CAP_HCI_PAGE_SCAN_REPETITION_MODE. ingroup btconfig
details Defines a default value of the page scan repetition mode when calling bt_l2cap_connect. Must be set to one of the following values: HCI_PAGE_SCAN_REPETITION_MODE_R0 HCI_PAGE_SCAN_REPETITION_MODE_R1 HCI_PAGE_SCAN_REPETITION_MODE_R2
The default value is HCI_PAGE_SCAN_REPETITION_MODE_R0. 
 
L2CAP_HCI_ROLE_SWITCH 
brief L2CAP_HCI_ROLE_SWITCH. ingroup btconfig
details Defines a default value of the role switch parameter when calling bt_l2cap_connect. Must be set to one of the following values: HCI_ROLE_SWITCH_ALLOW HCI_ROLE_SWITCH_DISALLOW The default value is to allow the role switch.
