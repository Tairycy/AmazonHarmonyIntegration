ADC Trigger Generation

The MCPWM module has the ability to trigger ADC at any instant within the PWM cycle without any software intervention. There are two module-level triggers and two channel-level triggers per channel.

Note: 
Please refer to the specific device data sheet for the exact count of available PWM channels that have the ability to generate ADC triggers. 

The available ADC triggers from the MCPWM module are as follows: 

MCPWM Module-level ADC Triggers:

  • Primary Special Event Trigger – Synchronized with Primary Master Time Base
  • Secondary Special Event Trigger – Synchronized with Secondary Master Time Base
Channel-level ADC Triggers:
  • Primary Trigger – Synchronized with local time base of the PWM channel.
  • Secondary Trigger – Synchronized with local time base of the PWM channel.
 

Configuring the MCPWM Module-level ADC Trigger

Use the following steps to set up MCPWM module-level ADC triggers:

  1. Disable the MCPWM module using PLIB_MCPWM_Disable
  2. Set the special event trigger instant and trigger post-scaler for Primary / Secondary Special Event trigger using PLIB_MCPWM_PrimarySpecialEventTriggerSetup / PLIB_MCPWM_SecondarySpecialEventTriggerSetup
  3. Enable the MCPWM module using PLIB_MCPWM_Enable
Example:

Copy Code
/*Disable the MCPWM Module*/
PLIB_MCPWM_Enable (MCPWM_ID_0);

/*Generate primary special even trigger when the primary master time base reaches 600 counts*/
/*Post scale the trigger output such that every second trigger is passed on to the ADC/interrupt controller*/
PLIB_MCPWM_PrimarySpecialEventTriggerSetup(MCPWM_ID_0, MCPWM_CHANNEL1, 600, 2);

/*Enable MCPWM Module*/
PLIB_MCPWM_Enable (MCPWM_ID_0);
Configuring the MCPWM Channel-level ADC Trigger

Use the following steps to set up MCPWM channel-level ADC triggers:

  1. Disable the MCPWM module using PLIB_MCPWM_Enable
  2. Set the following parameters of channel-level triggers using PLIB_MCPWM_ChannelTriggerSetup:
    • PWM Channel ID: ID of the PWM Channel whose triggers are being configured.
    • Trigger Post-Scaler: same parameter controls post-scaler for both Primary and Secondary triggers
    • Primary Trigger Cycle: Generate primary trigger while local timer is counting up/down/both.
    • Secondary Trigger Cycle: Generate secondary trigger while local timer is counting up/down/both.
    • ADC Trigger Source: ADC trigger asserted by primary trigger only / ADC trigger asserted by both primary and secondary triggers.
    • Trigger Interrupt Source: Interrupt is generated by Primary Trigger Event/ Secondary Trigger Event/ both.
    • Primary Trigger instant
    • Secondary Trigger instant
  3. Enable the MCPWM module using PLIB_MCPWM_Enable.
Example:

Copy Code
/*Disable the MCPWM Module*/
PLIB_MCPWM_Enable (MCPWM_ID_0);
/*Configure channel-level trigger for PWM Channel 1*/
/*Trigger prescaler = 4, i.e., Trigger output is asserted every fourth primary/secondary trigger event is asserted*/
/*Primary Trigger output is asserted for a primary trigger compare match event while the local timer is counting up*/
/*Secondary Trigger output is asserted for a secondary trigger compare match event while the local timer is counting up*/
/* ADC Trigger is asserted by both Primary and Secondary trigger output*/
/*Primary Trigger event generates an interrupt*/
/*Primary Trigger match occurs at 100 counts and Secondary Trigger match occurs at 500 counts*/
PLIB_MCPWM_ChannelTriggerSetup(MCPWM_ID_0 ,MCPWM_CHANNEL1, DIV_BY_4, COUNT_UP, COUNT_UP,
                               PRIMARY_SECONDARY, PRIMARY, 100, 500);
