bt_a2dp_get_stream_config Macro
C
#define bt_a2dp_get_stream_config(mgr, strm_handle) bt_avdtp_get_stream_config(mgr->avdtp_mgr, strm_handle)
Description

brief Get stream's configuration. ingroup a2dp 

details This function returns a pointer to a structure holding current configuration of the stream. 

param mgr A2DP manager. param strm_handle Stream handle. 

return li The stream's configuration if strm_handle specifies a valid stream and the stream is in one of the following state: 

AVDTP_STREAM_STATE_CONFIGURED AVDTP_STREAM_STATE_OPEN AVDTP_STREAM_STATE_STREAMING 

li NULL otherwise.
