TCPIP_MAC_PARAMETERS Structure
C
typedef struct {
  TCPIP_MAC_ADDR ifPhyAddress;
  TCPIP_MAC_PROCESS_FLAGS processFlags;
  TCPIP_MAC_TYPE macType;
  TCPIP_MAC_LINK_MTU linkMtu;
} TCPIP_MAC_PARAMETERS;
Description

MAC Run time parameters 

This is the data structure that the MAC user (TCP/IP stack) passes on to the MAC driver after the MAC initialization time to retrieve the settings resulted.

Members
Members 
Description 
TCPIP_MAC_ADDR ifPhyAddress; 
Physical address of the interface. MAC sets this field as part of the initialization process. 
TCPIP_MAC_PROCESS_FLAGS processFlags; 
MAC process flags. The stack will use this value to call into the MAC process function after receiving a MAC event. 
TCPIP_MAC_TYPE macType; 
MAC type: ETH, Wi-Fi, etc. 
TCPIP_MAC_LINK_MTU linkMtu; 
MAC link MTU size 
Remarks

None.
