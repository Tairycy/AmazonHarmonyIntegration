IPV6_ADDR_STRUCT Structure
C
typedef struct _IPV6_ADDR_STRUCT {
  struct _IPV6_ADDR_STRUCT * next;
  struct _IPV6_ADDR_STRUCT * prev;
  IPV6_ADDR address;
  unsigned long validLifetime;
  unsigned long preferredLifetime;
  unsigned long lastTickTime;
  unsigned char prefixLen;
  struct {
    unsigned char precedence;
    unsigned scope : 4;
    unsigned label : 4;
    unsigned type : 2;
    unsigned temporary : 1;
  } flags;
} IPV6_ADDR_STRUCT;
Description

IPv6 Address Structure 

This type defines all the fields of an IPv6 address.

Members
Members 
Description 
unsigned char precedence; 
Allow preferences 
unsigned scope : 4; 
Link-local, site-local, global. 
unsigned label : 4; 
Policy label 
unsigned type : 2; 
Uni-, Any-, Multi-cast 
unsigned temporary : 1; 
Indicates that the address is temporary (not public) 
Remarks

None.
