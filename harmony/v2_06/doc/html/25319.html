TCPIP_DNS_ENTRY_QUERY Structure
C
typedef struct {
  char* hostName;
  int nameLen;
  IPV4_ADDR * ipv4Entry;
  int nIPv4Entries;
  IPV6_ADDR * ipv6Entry;
  int nIPv6Entries;
  TCPIP_DNS_RESULT status;
  uint32_t ttlTime;
  TCPIP_NET_HANDLE hNet;
  int serverIx;
  int nIPv4ValidEntries;
  int nIPv6ValidEntries;
} TCPIP_DNS_ENTRY_QUERY;
Description

Structure: TCPIP_DNS_ENTRY_QUERY 

DNS module uses this structure to return information about a resolved IPv4 and IPv6 address.

Members
Members 
Description 
char* hostName; 
Pointer to a name to receive the host name for that particular entry 
int nameLen; 
hostName buffer size 
IPV4_ADDR * ipv4Entry; 
array of IPv4 entries/addresses to be populated 
int nIPv4Entries; 
number of entries in the ipv4Entry[] array; 
IPV6_ADDR * ipv6Entry; 
array of IPv6 entries/addresses to be populated 
int nIPv6Entries; 
number of entries in the ipv6Entry[] array; 
TCPIP_DNS_RESULT status; 
current status for this name:
  • TCPIP_DNS_RES_OK: name is resolved
  • TCPIP_DNS_RES_PENDING: name is pending
  • TCPIP_DNS_RES_SERVER_TMO: server timeout
 
uint32_t ttlTime; 
time to live for a solved DNS entry 
TCPIP_NET_HANDLE hNet; 
interface the name was obtained or on which the query is currently ongoing 
int serverIx; 
index of the server used on that interface 
int nIPv4ValidEntries; 
number of valid entries written to the ipv4Entry 
int nIPv6ValidEntries; 
number of valid entries written to the ipv6Entry 
Remarks

None.
