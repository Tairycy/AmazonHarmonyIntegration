TCPIP_IGMP_SOURCES_PER_GROUP Macro
C
#define TCPIP_IGMP_SOURCES_PER_GROUP 11
Description

maximum number of unicast sources per group a IGMP application can listen to Each unicast source can be used by multiple sockets Should be prime number for best performance

Remarks

TCPIP_IGMP_ASM_ALL_SOURCES may take a slot! 

Should be <= 32 

if TCPIP_IGMPV2_SUPPORT_ONLY is enabled then this value is ignored
