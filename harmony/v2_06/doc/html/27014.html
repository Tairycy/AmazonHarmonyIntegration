Library Overview

The library interface routines are divided into various sub-sections, which address one of the blocks of the overall operation of the Test Harness Library.

Library Interface Section 
Description 
System Interface Functions 
Functions used by the MPLAB Harmony system to initialize and maintain the Test Harness Library and libraries under test. 
Platform Initialization Functions 
Functions used to perform hardware or test platform-specific initialization. 
Test Results Functions 
Functions used by tests to report results to the harness. 
Test Time-out Functions 
Function used to maintain the test time out timer. 
Test Harness Initialization Data Types 
These structures provide the list of tests that the harness executes. 
Test Interface Function Pointer Types 
These pointers define the functions that the Test Harness requires each test must implement.
