SYS_RTCC_Initialize Function
C
SYS_MODULE_OBJ SYS_RTCC_Initialize();
Description

This function initializes the RTCC Service. It ensures that RTCC is stable and places the RTCC service in its initial state so that the service's state machine(s) can run.

Preconditions

None

Returns

SYS_MODULE_OBJ.

Remarks

This function must be called from the SYS_Initialize function. This function loops and blocks until RTCC is stable.

Example
Copy Code
sysObj.sysRtcc = SYS_RTCC_Initialize( );
