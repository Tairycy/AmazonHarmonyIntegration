Configuration-Bit Settings for PIC32MM0128GPM036

 

(Data Sheet # DS-60001387)  

Usage: 

#pragma config SETTING = VALUE 

 

 

SOSCHP 
Secondary Oscillator High Power Enable bit 
SOSCHP = ON 
SOSC oprerates in high power mode 
SOSCHP = OFF 
SOSC oprerates in normal power mode. 

 

ALTI2C 
Alternate I2C1 Pins Location Enable bit 
ALTI2C = ON 
Alternate I2C1 pins are used 
ALTI2C = OFF 
Primary I2C1 pins are used 

 

FUSBIDIO 
 
FUSBIDIO = ON 
USBID pin is controlled by the port function 
FUSBIDIO = OFF 
USBID pin is controlled by the USB module 

 

FVBUSIO 
 
FVBUSIO = ON 
VBUS pin is controlled by port function 
FVBUSIO = OFF 
VBUS pin is controlled by the USB module 

 

USERID 
User ID bits 
USERID = 00000000 
Range is from 0 to 0xffff 

 

JTAGEN 
JTAG Enable bit 
JTAGEN = ON 
JTAG is enabled 
JTAGEN = OFF 
JTAG is disabled 

 

ICS 
ICE/ICD Communication Channel Selection bits 
ICS = PGx1 
Communicate on PGEC1/PGED1 
ICS = PGx2 
Communicate on PGEC2/PGED2 
ICS = PGx3 
Communicate on PGEC3/PGED3 
ICS = NONE 
Not connected 

 

BOREN 
Brown-out Reset Enable bits 
BOREN = BOR3 
Brown-out Reset enabled in hardware; SBOREN bit disabled 
BOREN = BOR2 
Brown-out Reset enabled only while device active and disabled in Sleep; SBOREN bit disabled 
BOREN = BOR1 
Brown-out Reset controlled with the SBOREN bit setting 
BOREN = BOR0 
Brown-out Reset disabled in hardware; SBOREN bit disabled 

 

RETVR 
Retention Voltage Regulator Enable bit 
RETVR = ON 
Retention regulator is enabled and controlled by RETEN bit during sleep 
RETVR = OFF 
Retention regulator is disabled 

 

LPBOREN 
Downside Voltage Protection Enable bit 
LPBOREN = ON 
Low power BOR is enabled, when main BOR is disabled 
LPBOREN = OFF 
Low power BOR is disabled

RCLKSEL 
Run Mode Watchdog Timer Clock Source Selection bits 
RCLKSEL = SYSCLK 
Clock source is system clock 
RCLKSEL = FRC 
Clock source is FRC oscillator 
RCLKSEL = LPRC 
Clock source is LPRC (same as for sleep mode) 

 

FWDTEN 
Watchdog Timer Enable bit 
FWDTEN = ON 
WDT is enabled 
FWDTEN = OFF 
WDT is disabled 

 

FNOSC 
Oscillator Selection bits 
FNOSC = FRCDIV 
Fast RC oscillator (FRC) with divide-by-N 
FNOSC = PLL 
Primary or FRC oscillator with PLL 
FNOSC = PRI 
Primary oscillator (XT, HS, EC) 
FNOSC = DCO 
Digitally controlled oscillator (DCO) 
FNOSC = SOSC 
Low power secondary oscillator (SOSC) 
FNOSC = LPRC 
Low power RC oscillator (LPRC) 

 

PLLSRC 
System PLL Input Clock Selection bit 
PLLSRC = PRI 
Primary oscillator is selected as PLL reference input on device reset 
PLLSRC = FRC 
FRC oscillator is selected as PLL reference input on device reset 

 

SOSCEN 
Secondary Oscillator Enable bit 
SOSCEN = ON 
Secondary oscillator (SOSC) is enabled 
SOSCEN = OFF 
Secondary oscillator (SOSC) is disabled 

 

IESO 
Two Speed Startup Enable bit 
IESO = ON 
Two speed startup is enabled 
IESO = OFF 
Two speed startup is disabled 

 

POSCMOD 
Primary Oscillator Selection bit 
POSCMOD = EC 
External clock mode is selected 
POSCMOD = XT 
XT oscillator mode is selected 
POSCMOD = HS 
HS oscillator mode is selected 
POSCMOD = OFF 
Primary oscillator is disabled 

 

OSCIOFNC 
System Clock on CLKO Pin Enable bit 
OSCIOFNC = ON 
System clock is connected to CLKO/OSC2 pin 
OSCIOFNC = OFF 
OSCO pin operates as a normal I/O 

 

SOSCSEL 
Secondary Oscillator External Clock Enable bit 
SOSCSEL = OFF 
Crystal is used (RA4 and RB4 are controlled by SOSC) 
SOSCSEL = ON 
External clock is connected to SOSCO pin (RA4 and RB4 are controlled by I/O port registers) 

 

FCKSM 
Clock Switching and Fail-Safe Clock Monitor Enable bits 
FCKSM = CSDCMD 
Clock switching is disabled; Fail-safe clock monitor is disabled 
FCKSM = CSECMD 
Clock switching is enabled; Fail-safe clock monitor is disabled 
FCKSM = CSDCME 
Clock switching is disabled; Fail-safe clock monitor is enabled 
FCKSM = CSECME 
Clock switching is enabled; Fail-safe clock monitor is enabled

CP = OFF
