"persistent"

The persistent attribute indicates that the symbol cannot be modified by the user. 

Syntax: "persistent" ["if" <expr>] 

Example:  

Copy Code
config PERS
bool "Make persistent"
default y

config SOME_INT
int "Enter an int for $PROJECT_NAME in $DEVICE"
default 0
persistent if PERS
