plib_mcpwm.h

MCPWM Peripheral Library Interface Header 

This header file contains the function prototypes and definitions of the data types and constants that make up the interface to the MCPWM PLIB for all families of Microchip microcontrollers. The definitions in this file are common to the MCPWM peripheral.
