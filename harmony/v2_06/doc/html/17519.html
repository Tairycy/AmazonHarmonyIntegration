Library Overview

The library interface routines are divided into various sub-sections, which address one of the blocks or the overall operation of the CDAC module.

Library Interface Section 
Description 
Configuration Functions 
These functions are used to configure, enable, and disable the CDAC module. 
Data Control Functions 
These function are used to pass the data to converted to data registers and also read the data, if required. 
Feature Existence Functions 
These functions determine whether or not a particular feature is supported by the device.
