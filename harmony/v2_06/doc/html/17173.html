OPUS_DiskRead Function
C
int32_t OPUS_DiskRead(
    uint8_t * inBuff
);
Description

This function is called once to read one Opus packet.

Preconditions

None.

Parameters
Parameters 
Description 
*inBuff 
The pointer to the input buffer, where the Opus encoded data are to be written. 
Returns

This function returns the size of this Opus packet.

Example
Copy Code
int32_t bytesRead = OPUS_DiskRead(inputBuffer);
