PLIB_ETH_EVENTS Enumeration
C
typedef enum {
  ETH_EV_RXOVFLOW,
  ETH_EV_RXBUFNA,
  ETH_EV_TXABORT,
  ETH_EV_TXDONE,
  ETH_EV_RXACT,
  ETH_EV_PKTPEND,
  ETH_EV_RXDONE,
  ETH_EV_FWMARK,
  ETH_EV_EWMARK,
  ETH_EV_RXBUSERR,
  ETH_EV_TXBUSERR,
  ETH_EV_ALL
} PLIB_ETH_EVENTS;
Description

Ethernet Event Flags 

This enumeration defines flags for the possible Ethernet events that can cause interrupts.
