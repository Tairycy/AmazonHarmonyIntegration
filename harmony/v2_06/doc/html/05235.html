bt_avdtp_abort_stream Function
C
bt_bool bt_avdtp_abort_stream(
    bt_avdtp_mgr_t* mgr, 
    bt_byte strm_handle
);
Description

brief Suspend a stream. ingroup avdtp 

details This function tries to suspend a stream by sending a request to the remote party. The stream can be in any state state except AVDTP_STREAM_STATE_IDLE. As a result of this operation the AVDTP_EVT_ABORT_STREAM_COMPLETED event will be generated. This operation cannot be rejected. The p evt_param.abort_stream_requested.err_code is always == AVDTP_ERROR_SUCCESS. 

param mgr AVDTP manager. param strm_handle Stream handle. 

return li c TRUE if the function succeeds, i.e. the actual request has been sent to the remote party. li c FALSE otherwise. No events will be generated.
