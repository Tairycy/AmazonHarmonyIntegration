DRV_CAMERA_OVM7690_RECT Structure
C
typedef struct {
  uint32_t left;
  uint32_t top;
  uint32_t right;
  uint32_t bottom;
} DRV_CAMERA_OVM7690_RECT;
Description

OVM7690 Camera Window Rect 

This structure defines window rectangle co-ordinates as left, right, top, and bottom.

Members
Members 
Description 
uint32_t left; 
OVM7690 Camera Window left coordinate 
uint32_t top; 
OVM7690 Camera Window top coordinate 
uint32_t right; 
OVM7690 Camera Window right coordinate 
uint32_t bottom; 
OVM7690 Camera Window bottom coordinate 
Remarks

These values should be passed into the DRV_CAMERA_OVM7690_FrameRectSet function.
