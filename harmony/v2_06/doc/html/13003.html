GLCD_LAYER_ID Enumeration
C
typedef enum {
  GLCD_LAYER_ID_0 = 0x0,
  GLCD_LAYER_ID_1 = 0x1,
  GLCD_LAYER_ID_2 = 0x2,
  GLCD_LAYER_ID_MAX = 0x3
} GLCD_LAYER_ID;
Description

GLCD Layer Ids 

This data type defines the possible values of GLCD Layer Ids.

Members
Members 
Description 
GLCD_LAYER_ID_0 = 0x0 
GLCD Layer 0 Id 
GLCD_LAYER_ID_1 = 0x1 
GLCD Layer 1 Id 
GLCD_LAYER_ID_2 = 0x2 
GLCD Layer 2 Id 
GLCD_LAYER_ID_MAX = 0x3 
Maximum number of Layers supported by GLCD 
Remarks

This enumeration is processor specific and is defined in the processor-specific header files (see processor.h).
