bt_av_play_status_t Structure
C
typedef struct _bt_av_play_status_s {
  bt_av_response_t header;
  bt_ulong song_length;
  bt_ulong song_position;
  bt_byte play_status;
} bt_av_play_status_t;
Description

brief Parameter to AVRCP_EVT_GET_PLAY_STATUS_RECEIVED event ingroup avrcp 

details A pointer to this structure is passed to the AVRCP application callback as a valid member of the bt_avrcp_event_t union - bt_avrcp_event_t::play_status - when a local device received a response to a "get play status" request.
