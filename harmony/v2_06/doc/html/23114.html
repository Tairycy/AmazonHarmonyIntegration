SPEEX_Cleanup Function
C
void SPEEX_Cleanup();
Description

This function frees the memory that was allocated by the Speex Decoder and is called when the Speex Decoder ends.

Preconditions

None.

Returns

None.

Example
Copy Code
SPEEX_Cleanup();
