Building the Application

To build this project, you must open the usb_smart_speakerX project in MPLAB X IDE, and then select the desired configuration. 

The following tables list and describe the project and supported configurations. The parent folder for these files is: 

<install-dir>/apps/audio/usb_smart_speaker
