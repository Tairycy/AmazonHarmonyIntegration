Commands 

The following commands were used to generate the network performance numbers as measured with Iperf: 

UDP/IP Client:  

PIC32 Board 
Personal Computer 
iperf -c <PC IP Address> -u -b 100M -i 5 -t 20 
iperf -s -u -i 5 

UDP/IP Server:

PIC32 Board 
Personal Computer 
iperf -s -u -i 5 
iperf -c mchpboard_e -u -b 100M -i 5 -t 20 

TCP/IP Client:

PIC32 Board 
Personal Computer 
iperf -c <PC IP Address> -t 10 -i 5 -l 1024 -x 100M -M 1460 
iperf -s -i 5 -m -w 10K –N 

TCP/IP Server:

PIC32 Board 
Personal Computer 
iperf -s -i 5 -x 100M 
iperf -c mchpboard_e -t 10 -i 5 -l 1024
