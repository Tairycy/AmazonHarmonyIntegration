n2d_blend_t Enumeration
C
typedef enum n2d_blend {
  N2D_BLEND_NONE,
  N2D_BLEND_SRC_OVER,
  N2D_BLEND_DST_OVER,
  N2D_BLEND_SRC_IN,
  N2D_BLEND_DST_IN,
  N2D_BLEND_ADDITIVE,
  N2D_BLEND_SUBTRACT
} n2d_blend_t;
Description

Structure: n2d_blend 

N2D_BLEND_NONE - S, i.e. no blending N2D_BLEND_SRC_OVER - S + (1 - Sa) * D N2D_BLEND_DST_OVER - (1 - Da) * S + D N2D_BLEND_SRC_IN - Da * S N2D_BLEND_DST_IN - Sa * D N2D_BLEND_ADDITIVE - S + D N2D_BLEND_SUBTRACT - D * (1 - S)

Remarks

Some of the Nano2D API functions calls support blending. S and D represent source and destination color channels and Sa and Da represent the source and destination alpha channels
