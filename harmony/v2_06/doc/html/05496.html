bt_avrcp_mgr_callback_fp Type
C
typedef void (* bt_avrcp_mgr_callback_fp)(bt_avrcp_mgr_t* mgr, bt_byte evt, bt_avrcp_event_t* evt_param, void* callback_param);
Description

brief AVRCP application callback. ingroup avrcp 

details In order to be notified of various events a consumer of the AVRCP layer has to register a callback function (done with bt_avrcp_start()). The stack will call that function whenever a new event has been generated. 

param mgr AVRCP manager. 

param evt AVRCP event.
