SAMPLE_MODULE_INTERRUPT_MODE Macro
C
#define SAMPLE_MODULE_INTERRUPT_MODE false
Description

Sample Module, Interrupt Mode 

This macro determines whether or not the sample module will support interrupt interrupt driven execution. If defined as true, then the sample module will enable, disable, and clear the interrupt source identified in the SAMPLE_MODULE_INIT_DATA structure. Otherwise, it shoud be defined as false and the module will only support polled operation.
