SYS_CLK_SecondaryOscillatorEnable Function
C
void SYS_CLK_SecondaryOscillatorEnable();
Description

This function enables secondary oscillator which can be used as a clock source for peripherals like RTCC, Timer etc.. The SOSC requires a warm-up period of 1024 before it can be used as a clock source.

Preconditions

The SYS_CLK_Initialize function should be called before calling this API.

Returns

None.

Remarks

This function is supported on the following Microchip device families:

  • PIC32M

Example
Copy Code
SYS_CLK_SecondaryOscillatorEnable ( );
