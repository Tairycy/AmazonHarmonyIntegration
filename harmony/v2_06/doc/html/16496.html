MCPWM_FAULT_MODE Enumeration
C
typedef enum {
  MCPWM_FAULT_MODE_LATCHED,
  MCPWM_FAULT_MODE_CYCLE_BY_CYCLE,
  MCPWM_FAULT_MODE_DISABLED
} MCPWM_FAULT_MODE;
Description

MCPWM Channel Fault Mode 

This enumeration lists MCPWM Channel Fault Mode Configuration.

Members
Members 
Description 
MCPWM_FAULT_MODE_LATCHED 
MCPWM Channel Fault is configured in LATCHED Mode 
MCPWM_FAULT_MODE_CYCLE_BY_CYCLE 
MCPWM Channel Fault is configured in Cycle by Cycle Mode 
MCPWM_FAULT_MODE_DISABLED 
MCPWM Channel Fault is Disabled 
Remarks

None.
