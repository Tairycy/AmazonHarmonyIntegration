Module Operation

The CDAC module is based on a resistor network. The data is presented as the analog voltage output, based on the following equation.

The CDAC module outputs the analog voltage only when the module is enabled. The output of the CDAC module reaches the pin only if the Output Buffer is enabled.

Using the CDAC Module

The following code example shows how the CDAC module can be used. 

Example  

Copy Code
void CDACDemo{
//Initializing the data variable
uint16_t data = 3000;

//Disable the CDAC module
PLIB_CDAC_Disable( MY_CDAC_INSTANCE );

//Enable the CDAC Output
PLIB_CDAC_OutputEnable( MY_CDAC_INSTANCE );

//Select AVDD as Reference Voltage for resistor network
PLIB_CDAC_ReferenceVoltageSelect( MY_CDAC_INSTANCE,CDAC_REF_SEL_AVDD );

//Passing the data to be converted to data register
PLIB_CDAC_DataSet( MY_CDAC_INSTANCE, data );

//Enable the module
PLIB_CDAC_Enable( MY_CDAC_INSTANCE );
}
Note: 
Remember to wait TON time after enabling the CDAC module for the output to stabilize. Refer to the "Electrical Characteristic" chapter in the specific device data sheet for the TON specification.
