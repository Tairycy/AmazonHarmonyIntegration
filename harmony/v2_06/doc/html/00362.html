_bt_avctp_mgr_t Structure
C
struct _bt_avctp_mgr_t {
  bt_byte state;
  bt_byte flags;
  bt_avctp_channel_t* channels;
  bt_avctp_transport_t* transports;
  bt_avctp_mgr_callback_fp callback;
  void* callback_param;
  bt_avctp_channel_t* opening_channel;
};
Description

brief AVCTP manager. ingroup avctp 

details A structure that glues all pieces together. There is only one instance of this structure allocated by dotstack. A pointer to the instance can be get with c bt_avctp_get_mgr().
