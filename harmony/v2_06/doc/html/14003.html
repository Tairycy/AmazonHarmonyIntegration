Introduction

The Input System Service Library provides a simple API for the user to access input events of the underlying touch screen by using the configured input touch driver. The Input System Service also allows simple callback registration to notify the Aria Graphics library of input events. 

The Input System Service Library provides simple APIs to allow any input event provider (e.g. touch, keyboard, mouse, or application) to inject events into the input system. It also provides abstract callback listener registration which allows any interested listener within the application to receive events as they are broadcast. The Aria Graphics library is one such listener. 

The Input System Service replaces the Touch System Service in release v2.06.
