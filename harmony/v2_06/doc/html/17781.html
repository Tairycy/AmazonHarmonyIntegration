Library Overview

The library interface routines are divided into various sub-sections, which address one of the blocks or the overall operation of the MCPWM module

Library Interface Section 
Description 
MCPWM Module Control Functions 
These functions are used to enable/disable the MCPWM peripheral and control its behavior during Idle mode. 
Master Time Base Setup Functions 
These functions are used to configure the clock frequency prescaler and period value for the Primary/Secondary Master Time Base. 
Special Event Trigger Setup Functions 
These functions are used to configure the Primary/Secondary Special Event Triggers. 
Chop Clock Setup Functions 
These functions are used to configure Chop Clock frequency. 
PWM Generation Setup Functions 
These functions configure the time base source, duty cycle, phase shift, time base synchronization mode, alignment mode, output mode and dead-time for each PWM generator. 
PWM Output Control Setup Functions 
These functions control enable/disable, swap, polarity, and chopping of PWM outputs of each PWM generator. 
Trigger Generation Setup Functions 
These functions configure the trigger instant, postscaler and interrupts for primary/secondary channel triggers for each PWM generator 
Override Setup Functions 
These functions control enable/disable and output values for PWM output overrides. 
Fault Setup Functions 
These functions configure the fault input source, fault input polarity, fault mode and fault override output value for each PWM generator 
Current-Limit Setup Functions 
These functions configure the current-limit input source, current-limit input polarity, Current-Limit mode and current-limit override output value for each PWM generator. 
Leading Edge Blanking Setup Functions 
These functions configure the trigger sources and blanking period for leading edge blanking of fault and current-limit inputs. 
Period Match/Reset Interrupt Functions 
These functions manage (enable, disable, and clear) Period Match and Period Reset Interrupts.
