emWin_TouchInitialize Function
C
void emWin_TouchInitialize(
    const SYS_MODULE_INIT * const init
);
Description

This function initializes the emWin Touch Wrapper.

Preconditions

None.

Parameters
Parameters 
Description 
init 
Pointer to a data structure containing any data necessary to Initialize the emWin Touch Wrapper. 
Returns

None.

Remarks

None.

Example
Copy Code
EMWIN_TOUCH_INIT touchInit;

touchInit.iSysMsg = SYS_MSG_0;

emWin_TouchInitialize( &touchInit );
