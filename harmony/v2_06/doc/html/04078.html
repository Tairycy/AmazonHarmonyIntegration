Running the Demonstration

To run the AP-SCAN-EXAMPLE demonstration follows these steps:

  1. In MPLAB X IDE, load the project file wifi_winc1500_socket.X.
  2. Select the project configuration BSP.
  3. Edit the file app.h and set:
Copy Code
#define AP_SCAN_EXAMPLE    1
  1. Edit the file ap_scan.c and set the configuration for the example:
Copy Code
#if AP_SCAN_EXAMPLE
#define WLAN_SSID                   "DEMO_AP" /* Target AP */
#define WLAN_AUTH                   M2M_WIFI_SEC_WPA_PSK /* AP Security */
#define WLAN_PSK                    "12345678" /* Security Passphrase (If WPA Security Used) */
#define WLAN_WEP_KEY                "1234567890" /* Security Key (If WEP Security Used) */
#define WLAN_WEP_KEY_INDEX        1 /* Security Key Index (If WEP Security Used) */

#define PING_ADDRESS                "192.168.1.1" /* Address to Ping after Connection */
#define PING_COUNT                  3 /* Number of Times to Ping */
#define PING_INTERVAL               100 /* Wait 100ms between Pings */
  1. Build and run the demonstration.
  2. The example console outputs can be captured with TeraTerm, as shown in the following figure.
