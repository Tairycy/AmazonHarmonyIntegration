PLIB_MCPWM_ChopClockSetup Function
C
void PLIB_MCPWM_ChopClockSetup(
    MCPWM_MODULE_ID index, 
    uint16_t mcpwm_chop_clk_divider, 
    MCPWM_CHOP_CLOCK_CONTROL mcpwm_chop_clk_control
);
Description

This function sets up Chop Clock Generator which is used to chop the MCPWM output.

Preconditions

The MCPWM Module must be disabled before calling this function.

Parameters
Parameters 
Description 
index 
Identifier for the device instance to be configured 
mcpwm_chop_clk_divider 
The divider by which the MCPWM clock is divided to generate the Chop Clock i.e.Chop Clock = SYS_CLOCK/(2^PTCON)/CHOPCLOCK. 
Valid chop clock divider range 
2 - 1023. Minimum values is 2. A value of 0 or 1 will produce no chop clock
 
mcpwm_chop_clk_control 
Enables / Disables the MCPWM Chop Clock :MCPWM_CHOP_CLOCK_ENABLED MCPWM_CHOP_CLOCK_DISABLED 
Returns

None.

Remarks

This feature may not be available on all devices. Please refer to the specific device data sheet to determine availability or use PLIB_MCPWM_ExistsChopClockSetup in your application to determine whether this feature is available.

Example
Copy Code
// Where MY_MCPWM_INSTANCE, is the MCPWM instance selected for use by the
// application developer.
// Chop Clk = MCPWM Clock / 400.
// Chop Clk is enabled
PLIB_MCPWM_ChopClockSetup(MY_MCPWM_INSTANCE , 400 , MCPWM_CHOP_CLOCK_ENABLED);
