plib_cdac.h

Control Digital-to-Analog Converter (CDAC) Peripheral Library Interface Header 

This header file contains the function prototypes and definitions of the data types and constants that make up the interface to the CDAC PLIB for all families of Microchip microcontrollers. The definitions in this file are common to CDAC peripheral.

Functions
 
Name 
Description 
 
PLIB_CDAC_DataRead 
Returns the data on the CDAC register corresponding to the specified module. 
 
PLIB_CDAC_DataWrite 
Writes the specified data whose corresponding analog value to be output from CDAC. 
 
PLIB_CDAC_Disable 
Disables the CDAC module. 
 
PLIB_CDAC_Enable 
Enables the CDAC module. 
 
PLIB_CDAC_ExistsDataControl 
Identifies whether the DataControl feature exists on the CDAC module.
Implements pic32 variant of PLIB_CDAC_ExistsDataControl 
 
PLIB_CDAC_ExistsEnableControl 
Identifies whether the EnableControl feature exists on the CDAC module. 
 
PLIB_CDAC_ExistsOutputControl 
Identifies whether the OutputControl feature exists on the CDAC module. 
 
PLIB_CDAC_ExistsReferenceVoltageSelect 
Identifies whether the ReferenceVoltageSelect feature exists on the CDAC module. 
 
PLIB_CDAC_OutputDisable 
Disables the CDAC Output. 
 
PLIB_CDAC_OutputEnable 
Enables the CDAC Output. 
 
PLIB_CDAC_ReferenceVoltageSelect 
Selects the Reference Voltage that is applied to resistor network.
