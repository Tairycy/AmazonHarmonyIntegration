Introduction

The Control Digital-to-Analog Converter (CDAC) modules generate analog voltage corresponding to the digital inputs. The voltage can be used as a reference source for Comparators or can be applied to an Op amp. The CDAC module is targeted to generate reference or offset voltages for control applications, as opposed to other DAC modules; such as audio. 

Key features of this module include:

  • Wide voltage range (1.8V to 3.6V)
  • 12-bit resolution
