Building the Application

The WINC1500 socket demonstration contains many examples and only one example can be built and run at a time. The example can be selected from the app.h file. For example, to run the demonstration with the AP_SCAN_EXAMPLE, edit the app.h file and set:

Copy Code
 #define AP_SCAN_EXAMPLE    1

 

To build this project, you must open the wifi_winc1500_socket.X project in MPLAB X IDE, and then select the desired configuration. The following tables list and describe the project and supported configurations. The parent folder for these files is 

<install-dir>/apps/tcpip/wifi_winc1500_socket.
